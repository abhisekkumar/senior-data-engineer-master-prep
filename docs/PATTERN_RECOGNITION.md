# Pattern recognition

- Fast membership or counts: set/hash map.
- Contiguous range with monotonic validity: sliding window.
- Contiguous range with signed values or exact sums: prefix sum/map.
- Ordered pair constraints: two pointers.
- Monotonic decision boundary: binary search.
- Repeated next/previous greater item: monotonic stack.
- Top or bottom `k`: heap or bucket counting.
- Connectivity/dependencies: graph traversal or topological sort.
- Overlapping subproblems: dynamic programming.
