# Daily Plan

Generate today's plan with:

```bash
make daily
```

## Five required items

1. Due Python/DSA review
2. New or weak Python/DSA question
3. Python fundamentals, debugging, or SQL
4. Spark, data engineering, or system design
5. Behavioral, project explanation, GenAI, or troubleshooting

## End-of-session checklist

- [ ] Confidence before and after recorded
- [ ] Time spent recorded
- [ ] Hints recorded
- [ ] Mistakes recorded
- [ ] Time and space complexity stated
- [ ] Next review date generated
