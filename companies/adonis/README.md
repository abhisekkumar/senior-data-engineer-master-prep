# Adonis Interview Preparation

This directory organizes public, independently prepared study material for Adonis interviews. It is not affiliated with or endorsed by Adonis. The notes have been reformatted and generalized; they must not be treated as an account of confidential interview content.

Architecture-heavy notes include Mermaid diagrams that render directly on GitHub. Practice explaining each diagram from requirements through bottlenecks, failure modes, trade-offs, and operational recovery.

## Start here

1. Read the [study program](STUDY_PROGRAM.md).
2. Practice the [ask-and-think interview strategy](interview_strategy/ASK_AND_THINK.md).
3. Use the [consolidated phase checklist](checklists/CONSOLIDATED_PHASE_CHECKLIST.md).
4. Review the [coding preparation map](coding/README.md).
