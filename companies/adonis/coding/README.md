# Coding Preparation

Adonis coding preparation uses the shared [`leetcode_python/`](../../../leetcode_python/) library so solutions are not duplicated or allowed to drift.

Recommended progression:

1. Arrays, hashing, and prefix sums
2. Two pointers, intervals, and binary search
3. Sliding windows and stacks
4. Heaps and linked lists
5. Trees and graphs

Every numbered solution preserves the author's original implementation and surrounds it with clarifying questions, brute-force reasoning, pattern recognition, and complexity notes.
